## So what can a computer do?

_it can do math, it can store and retrieve information, it can display things on a screen, it can communicate with other computers_


## The web

_a way to link together content from any computer to any other computer_

## Program

_a set of instructions for a computer to follow_    

## Programs can… 

_do anything you can tell it to do, within the limits of the computer's capabilities_



* What is a web page? [a file that can be displayed in a web browser]

* What is a file format? [a way to organize data in a file so that a program can understand it]

* What is a web browser? [a program that can display content from the web]

* What is data? [information that a computer can store and manipulate]

* What is code? [the way you give a computer instructions]

* What is a file? [a way to store code or data on a computer]

* What is markup? [a way to tell a computer how to display content]

* What is a server? [a computer that stores and "listens" for requests for content]

* How do you add to the web? [by writing code and markup and putting it on a server]

* What's "webby" about the web? [links]
